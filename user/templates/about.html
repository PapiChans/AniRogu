About AniRogu

AniRogu is a mixed word of Anime and Log, when it translates to Japanese, the word "Log" turns into "Rogu (ログ)" That's why this Web Application can tracks the Episode of the Anime that you watching and also it can be categorized and sorted.

This Web Application is for Personal Use and to purpose to have my own tracker for myself also to enhance my programming skills and create this as a Unique Project on my own.

Features

The features of this Web Application is simple to create a personal anime log in your own. It includes the following:

  • Login
  • Sign Up
  • Account Recovery
  • Anime Search Engine
  • Profile Change Name
  • Change Password
  • Manage Anime List
  • Manage Episode List
  • Characters List

Account Recovery

It uses Simple Mail Transfer Protocol (SMTP) for Gmail. This requires to enter their own E-mail address and the Email Host Password in order to get the Account Recovery work. In the Settings.py there has already setup the SMTP and just put an E-Mail address and Host Password to setup Account Recovery Features.

Integration

To make the Search Engine works, this uses the Public API called "Jikan API" where it's unofficial API for MyAnimeList Website, Special Credits to them because I didn't make it if I didn't discover their API.
